# 🌍 Country Explorer

A modern Country Explorer web application built with **React.js**, **Axios**, **React Router DOM**, and **Tailwind CSS**.

This application allows users to explore countries from around the world, search countries instantly, view detailed country information, and learn interesting facts in a clean and interactive interface.

---

# 🚀 Features

- 🌎 Explore countries around the world
- 🔍 Search countries instantly
- 📄 View country details
- 👥 Population information
- 🏛️ Capital city information
- 🌍 Region details
- 🏳️ Country flags
- 📚 Interesting country facts
- 📖 About page
- 📩 Contact page with validation
- 📑 Pagination
- ⚡ Fast API integration using Axios
- 🔄 Client-side routing using React Router
- 🎨 Modern UI with Tailwind CSS
